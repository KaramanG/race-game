# race-game

Вторая практическая работа по дисциплине "Методика разработки бизнес-приложений средствами визуального программирования"

В рамках данной работы была реализована простая гоночная игра с использованием компонентов Script Machine и Graph. Версия редактора Unity - 6000.0.62f1.

Все модели сделал Даниил Бубнов.

Звуковые эффекты взяты с Mixkit: https://mixkit.co/free-sound-effects/

Музыка на фоне: Disturbed - Decadence
